🚀 Project OverviewAn advanced Full Stack Real-Time Support Chat Application engineered to facilitate instant, bidirectional communication between customers and support agents. The system utilizes WebSocket architecture to handle real-time message broadcasting, room-based chat isolation, and cross-origin synchronization with zero-lag user status updates.

✨Project scope (Features)
Instant Event Broadcasting: Low-latency, bidirectional message delivery powered by native WebSocket connections.Isolated Support Rooms: Multi-room functionality allows users to dynamically join unique chat channels or specific ticket IDs.Persistent Connection Logs: Active event listeners monitor user handshakes, clean room join events, and structural socket disconnections.Responsive Portal Interface: Clean, user-friendly UI designed for instant text delivery, display names, and local message timestamp generation.Auto-Scrolling Log View: Intelligently tracks the active dialogue viewport to keep the latest messages locked in full view.

🛠️ Tech Stack
Frontend: React, Tailwind CSS, JavaScript (ES6)
Backend: Node.js, Express.js
WebSocket Engine: Socket.io / socket.io-client
